# Shenqi Optimizer

《神泣》游戏自动化辅助工具 (学习用)

## 功能特性

- 🎮 **自动打怪** - 自动战斗、自动技能释放、自动吃药
- 🚀 **自动传送** - 快速移动、多地点循环
- 📦 **自动捡物** - 智能过滤、自动拾取
- 💊 **BUFF管理** - 自动维持增益效果
- ⚡ **战斗系统** - 技能冷却管理、状态处理
- 💰 **自动卖出** - 背包满自动卖出
- 🔧 **内存操作** - 直接游戏进程内存修改

## 技术架构

### 核心模块

```
ShenqiOptimizer/
├── Core/
│   ├── ConfigManager.cs       # INI 配置管理（基于参考项目格式）
│   ├── GameMemory.cs          # 游戏内存操作（模拟 asm.dll 功能）
│   ├── GameManager.cs         # 游戏进程管理
│   └── PerformanceMonitor.cs  # 性能监控
├── Features/
│   ├── AutoFarm.cs            # 自动打怪
│   ├── AutoTeleport.cs        # 自动传送
│   ├── AutoPickup.cs          # 自动捡物
│   ├── BuffManager.cs         # BUFF管理
│   ├── CombatSystem.cs        # 战斗系统
│   └── SellSystem.cs          # 自动卖出
├── UI/
│   └── MainForm.cs            # 主用户界面
├── Utils/
│   └── Logger.cs              # 日志系统
└── Config/
    ├── default.ini            # 默认配置
    └── patches.ini            # 内存补丁配置
```

## 配置系统

项目使用 **INI 配置文件驱动**，完全兼容参考项目的配置格式：

### default.ini 配置示例

```ini
[全局]
全局_编辑框_我的HP阈值=50
全局_编辑框_我的MP阈值=20
全局_选择框_自动传送=是
全局_编辑框_传送间隔=5000

[共用]
全局_过滤物品数组=
全局_被MS技能驱散的数组=
```

### patches.ini 配置示例

```ini
[补丁_登录1]
地址=00FACC61
数据=B801000000C9C3

[补丁_蓝屏暗桩]
地址=0041BD68
数据=C3
```

## 快速开始

### 系统要求

- Windows 7 或更高版本
- .NET 6.0 Runtime
- 管理员权限（用于内存操作）
- 《神泣》游戏已安装

### 安装与运行

1. 克隆仓库
```bash
git clone https://github.com/tiankongyiran/shenqi-optimizer.git
```

2. 打开项目
```bash
cd shenqi-optimizer
```

3. 使用 Visual Studio 编译运行
```bash
编译: Ctrl + Shift + B
运行: F5
```

4. **以管理员身份运行**程序

## 核心功能说明

### 自动打怪 (AutoFarm)

- 自动检测周围怪物
- 智能选择目标（距离/优先级）
- 自动释放技能序列
- 自动吃药/驱散负面状态
- 背包满自动卖出

### 自动传送 (AutoTeleport)

- 快速传送到指定位置
- 多地点循环传送
- 可配置传送间隔

### 自动捡物 (AutoPickup)

- 智能物品过滤
- 根据配置自动拾取
- 背包满提醒

### 战斗系统 (CombatSystem)

- 技能冷却管理
- 自动使用药物
- 负面状态检测和处理
- 队伍协作支持

### 内存操作 (GameMemory)

- 直接读写游戏进程内存
- 应用内存补丁
- 十六进制数据支持

## 配置参考

配置项基于参考项目的数据结构，支持以下类型：

- **布尔值**: `是`/`否`, `true`/`false`, `1`/`0`
- **整数**: 直接数字
- **字符串**: 文本内容
- **数组**: 用 `|` 分隔多个值

## 日志系统

日志输出到以下位置：

- **控制台**: 实时输出
- **文件**: `Logs/log_YYYYMMDD.txt`

日志格式: `[YYYY-MM-DD HH:mm:ss] [类型] 消息内容`
